# Kongming Dict Cleaner | 空明码词库清理工具

一个控制台程序，用于将满足特定条件的所有错误编码
从空明码词库文件`MasterDit.shp`中全部移除，提高词库
维护效率并避免错漏。具有轻量级、高性能的特点。

## 📥 用法

1. 配置C#环境，下载源码
2. 修改`Program.cs`：向`Proc`方法填入词库路径和用于查错的表达式，作为参数
3. 运行程序，清理后的词库文件将自动输出到相同目录中

### 查错方法

`Checker`类中提供了一系列查错方法，可根据条件选择：

| 方法名    | 依据   | 示例条件                   | 示例写法                                   |
|--------|------|------------------------|----------------------------------------|
| Char1  | 单字   | '冕'字首码不能为'M'           | Char1(code, word, '冕', 'M')            |
| Char2  | 单字   | '冕'字次码不能为's'           | Char2(code, word, '冕', 's')            |
| Char12 | 单字   | '冕'字前2码不能为"Ms"         | Char12(code, word, '冕', 'M', 's')      |
| Pair1  | 2字组合 | "石头"2字中，'石'字首码不能为'D'   | Pair1(code, word, '石', '头', 'D')       |
| Pair2  | 2字组合 | "石头"2字中，'石'字次码不能为'K'   | Pair2(code, word, '石', '头', 'K')       |
| Pair3  | 2字组合 | "石头"2字中，'头'字首码不能为't'   | Pair3(code, word, '石', '头', 't')       |
| Pair4  | 2字组合 | "石头"2字中，'头'字次码不能为'o'   | Pair4(code, word, '石', '头', 'o')       |
| Pair12 | 2字组合 | "石头"2字中，'石'字前2码不能为"Dk" | Pair12(code, word, '石', '头', 'D', 'k') |
| Pair34 | 2字组合 | "石头"2字中，'头'字前2码不能为"to" | Pair34(code, word, '石', '头', 't', 'o') |
